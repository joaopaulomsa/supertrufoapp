# Super Trufo App

MVP de jogo estilo Super Trunfo feito com React, Vite e TypeScript, preparado para publicar no GitHub Pages.

## Publicação

A aplicação é 100% frontend/static. Isso permite rodar diretamente no GitHub Pages sem backend.

URL esperada após configurar Pages:

```txt
https://joaopaulomsa.github.io/supertrufoapp/
```

## Rodar local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Modos do MVP

- Jogador contra IA
- Dois jogadores locais no mesmo navegador
- 20 cartas de personagens
- Escolha de atributo por rodada
- Placar e histórico

## Próxima fase

Para multiplayer online real com salas, será necessário adicionar backend externo, como Render, Railway ou Fly.io, pois GitHub Pages não executa Node.js/Socket.IO.
